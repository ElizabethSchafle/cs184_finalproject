26 /*
27  * Collada Viewer and helpful sub structures.
28  *
29  * Written by Bryce Summers on 9/14/2015
30  *
31  * 15-462 CMU Computer Graphics Assignment 2, Fall 2015.
32  *
33  * Purpose : This file specfies the behavior for a collada viewer.
34  *
35  * - The scene is rendered using OpenGl.
36  * - The user mouse and keyboard controls are mapped to viewing behaviors.
37  * - This class symbiotically interacts with the data structures
38  * from the collada parser and the geometric subdivision structures
39  * that the students will use in this glorious assignment!
40  *
41  *
42  * Picking (Feature Selection) Algorithm:
43  *
44  * 1. Find model space triangular coordinates and MeshFeature location
45  * data.
46  *
47  * 2. Project the model space triangular coordinates onto the screen with
48  * the current openGl model and projection matrices.
49  *
50  * 3. Solve 2 dimensional point on screen - triangle inclusion test.
51  * Note: This also gives us barycentric coordinates.
52  *
53  * 4. Find the triangle closest to the user using the homogenous w
54  * coordinate.
55  *
56  * 5. Determine whether the user selects the face ABC,
57  * edge AB, BC, CA, or vertice A, B, or C though the
58  * barycentric coordinates.
59  *
60  * 6. Select feature, show visual cues, and display debugging text.
61  */
